Прогнозирование коэффициентов восстановления золота из золотосодержащей руды
Российская компания, занимающаяся цифровизацией производства, разрабатывает решения для эффективной работы золотодобывающей отрасли, 
помогающие оптимизировать производство, чтобы не запускать предприятие с убыточными характеристиками.

Подготовить прототип модели машинного обучения для "Цифры"
Компания разрабатывает решения для эффективной работы промышленных предприятий.

Модель должна предсказать коэффициент восстановления золота из золотосодержащей руды. 
Модель поможет оптимизировать производство, чтобы не запускать предприятие с убыточными характеристиками.

Цель:

Подготовить данные;
Провести исследовательский анализ данных;
Построить и обучить модель.

Используемый технологический стек: NumPy, Pandas, Sklearn, MatPlotLib, Seaborn.

В ходе исследовательского анализа были проработаны пропуски, проанализированы столбцы каждого датасета. Так как для 
обучения и теста кол-во столбцов не совпадало, то пришлось уравнять их, так же пришлось добавить целевые столбцы, по колонке
с датой. Так же провели проверку правильности расчёта эффективного обогащения.
В процессе обучения моделей использовалась кроссвалидация. В качестве моделей обучения были выбраны: линейная регрессия, 
дерево решений и случайный лес. Лучшую метрику показал у нас случайный лес, его и использовали на тестовой выборке, получив
9.68 на внедренной метрике smape.
